# moguruu Design Notes

## Concept
moguruu は「潜る」から来ている。
AIやWeb技術を前面に出すのではなく、水面下で試したこと、拾った記録、作った仕組みを静かに残す場所。

## Site Purpose
moguruu.com is the quiet index for public experiments by usanari-tech.
It should make existing projects, notes, and logs discoverable without turning them into a sales page.
The site answers three questions:
- What exists now?
- What is being explored?
- Where can someone read or inspect more?

## Mood
- quiet
- archival
- observational
- minimal
- monochrome
- submerged

## Visual Rules
- Base colors are white, black, and gray.
- Accent colors appear only in photographic material.
- Avoid AI gradients, glowing objects, chat bubbles, robot motifs, and SaaS-like CTA blocks.
- Use lines, spacing, typography, and list structures as the main design language.

## Typography
- Japanese typography is a core part of the identity.
- Avoid rounded, pop, startup-like, and overly futuristic fonts.
- Prefer quiet, editorial, archival typefaces.
- Primary candidate: Noto Sans JP.
- Keep typography sans-serif only for now.
- Font should reduce AI/SaaS impressions and make the site feel like a personal archive.

## Copy Direction
- Avoid direct sales language.
- Use short, quiet labels.
- Prefer words like log, depth, archive, observation, lab, note.

## Layout Direction
- Top page behaves like an archive index.
- Hero should be simple and text-led.
- Logs should feel like records, not blog cards.
- Lab sections should feel like categories or shelves.
- Projects and notes should be concrete, visible, and linked.
